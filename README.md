# Verbel Bot on Telegram

Цей репозиторій містить бота Telegram для видавництва "Вербель".

## Основні можливості

- вибір мови (українська / англійська)
- перегляд каталогу книг
- надсилання повідомлень та файлів адміністраторам
- відповідь адміністраторам на запити користувачів
- бан користувачів та розсилка повідомлень від адміністратора

## Файли

- `main.py` — точка входу бота
- `config.py` — налаштування через `.env`
- `handlers/user.py` — обробка користувачів та повідомлень
- `handlers/admin.py` — команди адміністратора
- `bot/dispatcher.py` — реєстрація маршрутизаторів
- `keyboards/user_kb.py` — клавіатури для користувачів
- `data/users.json` — збереження зареєстрованих та забанених користувачів

## Параметри середовища

Створіть `.env` у корені проекту з такими значеннями:

```env
BOT_TOKEN=ваш_токен
ADMIN_IDS=123456789,987654321
```

- `BOT_TOKEN` — Telegram Bot API токен
- `ADMIN_IDS` — список ID адміністраторів через кому

## Команди для користувачів

- `/start` — запуск бота та вибір мови
- `/help` — посилання на сайт
- кнопка `📚 Каталог` — перегляд доступних книг
- кнопка `✍️ Написати` — відправити текст адміністратору
- кнопка `🎥 Відправити медіа` — відправити файл адміністратору

## Адмінські команди

Команди працюють тільки для ID з `ADMIN_IDS`.

- `/ban <user_id>` — забанити користувача
- `/unban <user_id>` — розбанити користувача
- `/broadcast <повідомлення>` — розіслати повідомлення всім зареєстрованим незаблокованим користувачам

## Робота з ботом

Запустіть бота через Python:

```bash
python main.py
```

## Примітки

- Для роботи збереження користувачів та банів використовується файл `data/users.json`.
- Заблоковані користувачі не можуть взаємодіяти з ботом і отримують повідомлення про бан.
- Файли конфігурації `.env` та директорії `data/` і `logs/` не повинні потрапляти в публічний репозиторій.

## Безпека

- Не публікуй `.env` у GitHub, там зберігається токен бота.
- Не розміщуй `data/users.json` у відкритому доступі на вебсервері.
- Якщо `.gitignore` створено, то `data/`, `logs/` і `.env` будуть ігноровані.
